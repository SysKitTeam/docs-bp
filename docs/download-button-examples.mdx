# Download Button Examples

You can use the custom DownloadButton component in any MDX file like this:

```mdx
import DownloadButton from '@site/src/components/DownloadButton';

## PowerShell Scripts

<DownloadButton 
  href="/img/get-bpapppoolgroupmembership.zip"
  filename="get-bpapppoolgroupmembership.zip"
  fileSize="2.3 KB"
  fileType="ZIP"
  description="PowerShell script to get application pool group membership"
/>

<DownloadButton 
  href="/img/get-bpdbstatus.7z"
  filename="get-bpdbstatus.7z"
  fileSize="1.8 KB"
  fileType="7Z"
  description="PowerShell script to check database status"
/>

## Configuration Files

<DownloadButton 
  href="/img/set-bploopbackcheckconfig.zip"
  filename="set-bploopbackcheckconfig.zip"
  fileSize="1.5 KB"
  fileType="ZIP"
  description="Script to configure loopback check settings"
/>
```

## Available Props

- `href`: The download URL (required)
- `filename`: Display name of the file (required)
- `fileSize`: Human-readable file size (required)
- `fileType`: File type/extension for icon and badge (required)
- `description`: Optional description text

## File Type Icons

The component automatically selects icons based on file type:
- 📦 ZIP, 7Z files
- 📄 PDF files
- 📝 DOC, DOCX files
- 📊 XLS, XLSX files
- 📈 PPT, PPTX files
- ⚡ PowerShell files
- 📁 Default for other files
